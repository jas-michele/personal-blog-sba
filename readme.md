# 1. Brief description of your project

This project is a personal blog application that allows users to create, edit, and delete blog posts. Posts are added through a form and displayed dynamically on the page. The application uses localStorage to save and retrieve posts so they persist after refreshing.

# 2. Instructions to run the application

Download or clone the project and open the index.html file in a browser. No additional setup is required.

# 3. Reflection (process, challenges, solutions)

During this project, I focused on working with the DOM and handling user input through forms. I learned how to dynamically create, update, and remove elements based on user action.

One challenge I faced was implementing the edit and delete functionality, I had to make sure I was targeting the correct post and updating both the DOM and localStorage so the changes stayed consistent

I also used console.log() to debug issues and track values when things were not displaying as expected.
